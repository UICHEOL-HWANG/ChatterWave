# 웹 구축 특이점 

# 1. 🔑 JWT를 이용한 회원 보안 강화 

- 사이트 이동중 -> 웹소켓 등 여러 기능들을 이용중에 회원 보안을 위한 토큰 감별 사용 

# 2. 차단된 회원 사후처리 
- `3회 경고` 이후 혐오 표현 사용자 차단하여 채팅 라우터를 일정시간 접근하지 못하도록 설정 

# 3. 메세지 캐싱 
- 모델을 통한 메세지 혐오 유무 확인으로 인해 API 요청을 2번 사용 → 이로 인한 채팅서비스 속도 저하 

- 메세지 캐싱을 통해 약 3초 정도 속도 완화
